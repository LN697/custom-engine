# Custom Engine Documentation

Welcome to the custom 3D engine documentation. This repository is structured to support a modular engine architecture with distinct subsystems for input, rendering, and platform integration.

## Contents

- `docs/getting_started.md` — Setup, build, and run instructions.
- `docs/architecture.md` — High-level design and engine layers.
- `docs/modules.md` — Module responsibilities and code organization.
- `docs/dependencies.md` — Required packages and installation guidance.

## Goals

- Keep engine subsystems isolated and testable.
- Provide a clear path for expanding rendering, asset loading, and scene management.
- Make Linux development simple with `Makefile` automation.
